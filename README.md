## 🚀 Introduction

A modern, responsive developer portfolio built using **React**, **TailwindCSS**, and **Vite**.  
Designed to showcase projects, technical skills, and real-world problem-solving with a clean UI and smooth user experience.

---

## ⚙️ Tech Stack

* **React** – Component-based UI development  
* **Vite** – Fast build tool  
* **TailwindCSS** – Utility-first styling  
* **Lucide Icons** – Clean icon set  
* **Radix UI** – Accessible UI components  
* **TypeScript (optional)** – Type safety  
* **GitHub & Vercel** – Deployment  

---

## ⚡️ Features

* 🌑 **Light/Dark Mode Toggle**  
* 💫 **Smooth Animations & Effects**  
* 📱 **Fully Responsive Design**  
* 👨‍💻 **Hero & About Sections**  
* 📊 **Skills Visualization**  
* 🖼️ **Project Showcase**  
* 📩 **Contact Section**  
* 🚀 **Easy Deployment**

## 👌 Quick Start

### Prerequisites

* [Node.js](https://nodejs.org/)  
* [Git](https://git-scm.com/)  

### Clone and Run

```bash
git clone https://github.com/Pradumnsaindane/pradumn-saindane-portfolio.git
cd pradumn-saindane-portfolio
npm install
npm run dev
